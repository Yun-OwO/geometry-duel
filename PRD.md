# 几何决斗 HTML5 复刻 - 需求文档

## 1. 项目概述

| 项目 | 内容 |
|------|------|
| 项目名称 | Geometry Duel (几何决斗) |
| 项目类型 | 2D 动作竞技游戏 |
| 核心功能 | 1v1 极简几何风格弓箭对战 |
| 目标平台 | Web (PC & Mobile) |

## 2. 游戏简介

复刻自 TapTap 游戏《几何决斗》(https://tap.cn/lljXFkRCp)，原项目基于 OpenProcessing 开源草图，由 pama1234 移植至 Java LibGDX 引擎。本项目为 HTML5 版本，使用 PixiJS 引擎重写。

## 3. 玩法说明

### 3.1 核心机制

- **游戏模式**: 1v1 人机对战
- **胜负条件**: 击杀对方玩家

### 3.2 操作方式

#### 键盘操作 (PC)
| 按键 | 功能 |
|------|------|
| Z | 普通攻击 |
| X | 终极技能 |
| W/↑ | 向上移动 |
| S/↓ | 向下移动 |
| A/← | 向左移动 |
| D/→ | 向右移动 |
| P | 暂停/继续 |

#### 触控操作 (Mobile)
| 区域 | 功能 |
|------|------|
| 左半屏 | 虚拟摇杆移动 |
| Z区域(右下) | 普通攻击 |
| X区域(左下) | 终极技能 |

### 3.3 技能说明

#### 普通攻击 (Z键)
- **射速**: 0.2秒/发
- **瞄准**: 自动追踪敌人方向
- **效果**: 击退敌人并造成约0.75秒眩晕
- **子弹**: 可被敌方子弹抵消
- **移动惩罚**: 发射时移动速度降低

#### 终极技能 (X键)
- **蓄力时间**: 0.5秒
- **瞄准方式**: 方向缓慢旋转瞄准
- **效果**: 一击必杀
- **可被打断**: 被普攻命中时中断
- **移动惩罚**: 蓄力时移动速度降低

## 4. 角色设计

### 4.1 玩家
- **外观**: 白色正方形
- **大小**: 32x32 逻辑像素
- **初始位置**: 画布底部中央

### 4.2 AI对手
- **外观**: 黑色正方形
- **大小**: 32x32 逻辑像素
- **初始位置**: 画布顶部中央

### 4.3 子弹设计
| 类型 | 外观 | 速度 | 特效 |
|------|------|------|------|
| 普攻子弹 | 短小线条 | 24 | 白色/黑色 |
| 大招子弹 | 长线+箭头 | 64 | 红色高亮 |

## 5. 屏幕适配

### 5.1 支持比例
| 比例 | 说明 |
|------|------|
| 16:9 | 标准宽屏 |
| 21:9 | 超宽屏/带鱼屏 |
| 4:3 | 经典比例 |
| 9:16 | 竖屏手机 |
| 任意比例 | 自适应填充 |

### 5.2 适配策略
- **逻辑分辨率**: 640x640 正方形
- **渲染策略**: 视角场拓展，根据屏幕比例动态调整视口
- **保证事项**:
  - 无黑边
  - 不裁切游戏角色
  - 视野开阔

### 5.3 高DPI支持
- `autoDensity: true`
- `resolution: window.devicePixelRatio`
- 保证高分辨率设备显示清晰

## 7. UI设计

### 7.1 风格
- 极简几何风格
- 无多余装饰
- 保持与游戏画面一致

### 7.2 暂停菜单
- 几何形状按钮
- 简洁文字提示
- 半透明背景遮罩

## 8. 性能要求

- 目标帧率: 60 FPS
- 响应延迟: < 16ms
- 移动端触控延迟优化
